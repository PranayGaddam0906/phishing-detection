# 🔒 Phishing Website Detection

A machine learning project to detect phishing websites using **KNN**, achieving 95% accuracy.  
Deployed with **Streamlit** for an interactive web app.

## 🚀 Features
- Detects phishing websites in real-time
- Uses feature extraction + ML (KNN)
- Interactive UI with Streamlit
